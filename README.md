# Aplikacja wypożyczalni samochodów

Aplikacja wypożyczalni samochodów od strony firmy wypożyczającej.

Baza danych: samochod, klient, zlecenie, slownik, niedostepnosc*
samochod: ID, marka, model, rok, moc, ile osob, typ, czy usunięty, udogodnienia*
klient: ID, imie, nazwisko, nazwa, pesel, nip, nr telefonu, prawo jazdy
zlecenie: ID, klientID, samochodID, data od, data do, ilosc, typ stawki, stawka, kwota
slownik: ID, typ ID, nazwa, opis

Funkcjonalności aplikacji: 
- wypożyczenie
- dodanie klienta
- dodanie/usunięcie samochodu
- dodanie niedostępności*
